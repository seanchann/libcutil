# libcutil

基于asterisk等优秀开源项目源码编写，提供一些基础的C语言库，来辅助C/C++类的开发，目前支持：

- console
- cli
- log
- http mini server
- restful mini framework
- 公共基础库，例如：list以及系统函数的包装
